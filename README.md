# Personal Finance Scenario Simulator

## Overview
Interactive finance planning tool for testing monthly budget scenarios and savings outcomes.

Website demo:- https://personal-finance-scenario-simulator.vercel.app/

This project is designed for portfolio demos and recruiter walkthroughs:
- clear input panel
- visual KPI cards
- goal progress bar and budget breakdown
- practical edge-case handling

## Core Features
- Monthly savings calculation and savings-rate grading
- Emergency fund runway and target coverage check
- Savings goal timeline estimate (up to 30-year horizon)
- Projection view for the next 3 years
- Guardrails for zero-income, negative cash-flow, and already-reached goal scenarios

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript

## Local Development
- `npm run build`
- `npm test`
- `npm run preview`
